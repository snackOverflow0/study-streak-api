# Study Streak API 🚀

A backend API that tracks user study habits and streaks (like Duolingo).

## Features
- User creation
- Study logging
- Automatic streak calculation
- Longest streak tracking

## Tech Stack
- Node.js
- Express.js
- PostgreSQL (Neon)

## Endpoints

### Users
POST /users  
GET /users/:id  

### Logs
POST /logs  
GET /logs/:userId
